# FreeCAD Mechanical Design Portfolio

This repository showcases CAD models and parametric design experiments created using FreeCAD. The focus of these projects is mechanical component modeling, parametric workflows, and digital asset preparation for engineering simulations and manufacturing concepts.

## Tools Used
- FreeCAD
- Parametric Modeling
- Mechanical Part Design
- Sketch Constraints
- Assembly Modeling
- Technical Drawing Generation


## Project 1: Parametric Gear Assembly

A parametric gear model designed to demonstrate mechanical relationships between rotating components.

Features:
- Parametric gear dimensions
- Constraint-based sketch modeling
- Mechanical alignment testing
- Assembly positioning



## Project 2: Modular Bracket System

A structural bracket designed using FreeCAD's parametric workflow.

Features:
- Adjustable mounting points
- Constraint-driven geometry
- Mechanical reinforcement design
- 3D model visualization



## Project 3: Mechanical Housing Prototype

A prototype enclosure model used to test component fitting and structural layout.

Features:
- Parametric wall thickness
- Internal component mounting
- Structural rib support
- CAD assembly layout


These models demonstrate the use of FreeCAD for parametric CAD design, mechanical prototyping, and digital asset preparation used in engineering and simulation workflows.
